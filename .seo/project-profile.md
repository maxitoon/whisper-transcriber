# Project SEO Profile

## Project Name

Whisper Transcriber

## One-line Description

Local Whisper speech-to-text transcriber for private audio, video, YouTube, Zoom, and voice-note transcription with Python and whisper.cpp.

## Long Description

Whisper Transcriber is a local-first audio and video transcription tool built around `whisper-cli` from whisper.cpp, plus an optional Python CLI. It helps creators, researchers, journalists, developers, and privacy-conscious users convert recordings into text, subtitles, or JSON without relying on hosted transcription services.

The project focuses on practical workflows: live microphone transcription, YouTube download and transcription, Zoom recordings, WhatsApp audio, and arbitrary local files. It supports multiple Whisper models and output formats including txt, srt, vtt, and json.

## Target Audience

- Primary: creators, researchers, journalists, developers, and knowledge workers who need local transcription
- Secondary: privacy-conscious users who want to avoid uploading sensitive recordings to cloud transcription APIs
